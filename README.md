# Projetos de estudo em Aprendizagem de máquina

Neste repositório você poderá encontrar alguns exercícios de aprendizagem de máquina que foram utilizados para estudo. A maioria dos conteúdos são de nível básico ou intermediário.

Os conteúdos estão organizados em diretórios separados por tema. Cada diretório será possível encontrar tudo o que foi utilizado para executar o experimento para que seja possível replicar os testes e simulações.
